# Website Redesign Update

Replace these two files in the root of the GitHub repository:

- `app.py`
- `README.md`

## GitHub browser steps

1. Open `app.py` in the repository.
2. Click the pencil icon.
3. Select all existing text and replace it with the new `app.py`.
4. Commit the change.
5. Repeat for `README.md`.

An easier alternative is to use **Add file → Upload files** and upload both
replacement files. Confirm that GitHub shows them as changed files, then commit.

Streamlit Community Cloud should redeploy automatically after the commit.
