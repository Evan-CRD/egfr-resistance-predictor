# EGFR Structure–Function Drug Response Website

Version 6 adds:

- Experimental response values and mutation-held-out predictions on the Prediction page.
- A Dataset Source page describing the 2021 Nature paper, experimental screen, and structure-function classification.
- A detailed pipeline page based directly on the training and prediction scripts.
- Download buttons for the source workbook, cleaned dataset, and mutation-specific prediction tables.
- Automatic locking of each mutation to its published structural group.

## Deploy

Upload every file and folder to the root of the GitHub repository. In
Streamlit Community Cloud, use `app.py` as the main file and reboot the app.

This is an experimental research tool and not clinical advice.
