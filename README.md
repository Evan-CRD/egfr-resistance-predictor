# EGFR Structure–Function Drug Response Website

This version includes:

- Experimental and mutation-held-out predicted values on the Prediction page.
- A concise Dataset Source page citing the 2021 Nature paper and providing the data files.
- A simplified visual pipeline with expandable explanations for:
  1. dataset construction,
  2. replicate medians,
  3. five candidate feature sets,
  4. CatBoost regression,
  5. mutation-held-out validation.

## Deploy

Upload every file and folder to the root of the GitHub repository. Use
`app.py` as the Streamlit main file, then reboot the application.
